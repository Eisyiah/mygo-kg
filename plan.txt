# MyGO Knowledge Graph Archive

## Vision

Build the most complete public knowledge archive for:

* BanG Dream! It's MyGO!!!!!
* Ave Mujica

The project is not a traditional wiki.

The project is a knowledge graph archive where every piece of content, paragraph, sentence, character, song, event, and concept becomes a searchable and interconnected node.

Users should be able to:

* Read source materials
* Search at document, paragraph, and sentence level
* Browse relationships
* Explore timelines
* Discover related content automatically
* Traverse the entire knowledge network

Deployment target:

* GitHub Pages
* Static site generation
* No paid services
* No backend required
* Long-term maintainability

---

# Core Design Philosophy

Everything is a node.

Characters are nodes.

Songs are nodes.

Events are nodes.

Documents are nodes.

Paragraphs are nodes.

Sentences are nodes.

Tags are nodes.

The archive should function as a knowledge network rather than a collection of pages.

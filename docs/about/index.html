About

The MyGO Knowledge Graph Archive is a searchable, interconnected knowledge network for BanG Dream! It's MyGO!!!!! and Ave Mujica.

How It Works

Every piece of content — characters, songs, events, documents, paragraphs, and sentences — becomes a node in the knowledge graph. Nodes are connected through relationships, allowing you to traverse the entire knowledge network without reaching dead ends.

Content Sources

  • Anime Episodes
  • Game Stories
  • Official Interviews
  • Official Radio
  • Official Blog Posts
  • Official Event Reports
  • Live Reports
  • Magazine Interviews
  • Staff Interviews
  • Voice Actor Interviews

Technology

  • Built with Astro
  • Graph visualization by Cytoscape.js
  • Search powered by FlexSearch
  • Hosted on GitHub Pages
  • Static site generation — no backend required

Contributing

Content is stored in Git. Contributors can add markdown files, edit metadata, add annotations and tags, and submit Pull Requests. The site rebuilds automatically.
